Comandos más utilizados de Git y Navegación en Carpetas

Configuración

Configurar el nombre y correo electrónico del usuario:

      
        git config --global user.name "Tu Nombre"
        git config --global user.email "tu@email.com"

Creación y clonación de repositorios

Inicializar un repositorio vacío en la carpeta actual:

      
        git init

Clonar un repositorio remoto:

      
        git clone <url_repositorio>

Registro de cambios

Ver el estado de los archivos en el repositorio:

      
        git status

Agregar cambios al área de preparación:

      
        git add <archivo>

Crear un nuevo commit con los cambios agregados:

      
        git commit -m "Mensaje del commit"

Exploración del historial

Ver el historial de commits:

      
        git log

Ver el historial de commits de forma simplificada:

      
        git log --oneline

Ver las diferencias entre commits o cambios sin preparar:

      
        git diff

Ramas

Ver las ramas existentes en el repositorio:

      
        git branch

Crear una nueva rama:

      
        git branch <nombre_rama>

Cambiar a una rama existente:

      
        git checkout <nombre_rama>

Combinar los cambios de una rama en otra:

      
        git merge <nombre_rama>

Colaboración

Agregar un repositorio remoto:

      
        git remote add origin <url_repositorio_remoto>

Enviar los cambios locales a un repositorio remoto

            
              git push origin <nombre_rama>

Obtener los cambios del repositorio remoto y fusionarlos en la rama local: 

            
              git pull origin <nombre_rama>

Navegación en Carpetas

Ver el contenido de una carpeta:

            
              ls

Cambiar al directorio anterior:

            
              cd ..

Ingresar a una carpeta específica:

            
              cd nombre_carpeta

Ver la ruta de la carpeta actual:

            
              pwd

Manipulación de Archivos y Carpetas

Crear una carpeta:

    
      mkdir nombre_carpeta

Crear un archivo:

    
      touch nombre_archivo

Eliminar un archivo:

    
      rm nombre_archivo

Eliminar una carpeta:

    
      rm -r nombre_carpeta
